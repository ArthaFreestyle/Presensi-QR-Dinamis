Dashboard Information

Real-time Monitoring:

Select a device ID and click "Start Polling" to continuously monitor accelerometer readings from the selected device.

Data Retention:

The chart displays up to 50 most recent data points. New readings are automatically added as they arrive.

Update Frequency:

Data is fetched every 2-3 seconds from the backend API endpoint ?path=telemetry/accel/latest.
